# Mid-Term-Project

hw6 will be our course **midterm project**.

A few instructions:

1. First, clone this repository to your computer
2. Then, make sure you activate the Conda environment before starting the jupyter notebook
3. In your terminal, run the following commands:
  ```console
    conda activate cse2107
    jupyter notebook
  ```
4. For your data source follow the link provided in the hw notebook. Be sure to save the data under the data folder in the "utility" directory.

## Data Information
Where the data comes from:

The dataset used in this project should be downloaded from: County Health Rankings and Roadmaps

https://www.countyhealthrankings.org/health-data/methodology-and-sources/data-documentation 

The data has been collected from different sources, more information can be found here 

https://www.countyhealthrankings.org/what-impacts-health/county-health-rankings-model 

For the purposes of this project, download the the data (2024 CHR CSV Analytic Data) from https://www.countyhealthrankings.org/health-data/methodology-and-sources/data-documentation 

The original dataset (analytic_data2024.csv) had 3195 rows and 770 columns.
